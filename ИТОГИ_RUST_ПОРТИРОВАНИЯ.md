## 🔧 Технические детали

### Архитектура

**Паттерн**: State Machine + Trait Objects

```rust
// Трейт для всех сцен
trait Scene {
    fn handle_input(&mut self);
    fn update(&mut self);
    fn draw(&self);
    fn get_next_state(&self) -> Option<GameState>;
}

// Использование
let mut current_scene: Box<dyn Scene> = Box::new(MenuScene::new());
```

### Зависимости

```toml
macroquad = "0.4"      # Игровой движок (OpenGL)
serde = "1.0"          # Сериализация
serde_json = "1.0"     # JSON
rand = "0.8"           # Генератор случайных чисел
```

### Оптимизация

```toml
[profile.release]
opt-level = 3          # Максимальная оптимизация
lto = true             # Link Time Optimization
codegen-units = 1      # Лучшая оптимизация (медленнее компиляция)
strip = true           # Удаление отладочной информации
```

## 🚀 Как использовать

### Установка Rust

**Windows:**
```bash
winget install Rustlang.Rustup
```

**Linux/Mac:**
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### Сборка

**Автоматически:**
```bash
BUILD_RUST.bat
```

**Вручную:**
```bash
# Разработка (быстро)
cargo build
cargo run

# Релиз (оптимизировано)
cargo build --release
./target/release/game.exe
```

### Запуск

```bash
# Из исходников
cargo run --release

# Скомпилированный exe
./target/release/game.exe
# или
./AdventurePuzzleGame_Rust.exe
```

## ✨ Особенности

- 🎯 **6 уникальных уровней** с различными типами головоломок
- 🏘️ **Город-хаб** для выбора уровней и взаимодействия с NPC
- 🎲 **3 типа мини-игр**: угадай число, камень-ножницы-бумага, викторина
- 🎭 **Система визуальной новеллы** с диалогами и портретами персонажей
- 💰 **Система инвентаря** с золотом и предметами
- 💾 **Автосохранение** прогресса в JSON
- 🎨 **Красивое анимированное меню** с эффектами
- 📦 **Сборка в .exe** одним кликом

## 🚀 Быстрый старт

### Для игроков

1. Скачайте последний [релиз](https://github.com/lololoevish/pazzle-game/releases)
2. Запустите `AdventurePuzzleGame.exe`
3. Играйте!

### Для разработчиков

```bash
# Клонировать репозиторий
git clone https://github.com/lololoevish/pazzle-game.git
cd pazzle-game

# Установить зависимости
pip install -r requirements.txt

# Запустить игру
python src/main.py

# Собрать в exe
pyinstaller AdventurePuzzleGame.spec
```

## 🎮 Управление

| Действие | Клавиши |
|----------|---------|
| Движение | ↑↓←→ или WASD |
| Выбор в меню | ↑↓ или W/S |
| Подтверждение | Enter или Пробел |
| Взаимодействие | Мышь |
| Назад/Выход | ESC |

## 📁 Структура проекта

```
├── src/                       # Исходный код
│   ├── main.py               # Точка входа
│   ├── scenes/               # Игровые сцены
│   │   ├── menu.py          # Главное меню
│   │   ├── town.py          # Город-хаб
│   │   ├── gameplay.py      # Игровой процесс
│   │   ├── minigames.py     # Мини-игры
│   │   └── story.py         # Сюжетные сцены
│   ├── entities/             # Игровые объекты
│   │   ├── player.py        # Игрок
│   │   └── item.py          # Предметы
│   └── utils/                # Утилиты
│       ├── helpers.py       # Вспомогательные функции
│       ├── visual_effects.py # Визуальные эффекты
│       ├── save_system.py   # Система сохранений
│       └── dialogue_system.py # Система диалогов
├── dist/                     # Готовый exe (после сборки)
├── docs/                     # Документация
└── assets/                   # Ресурсы (звуки, спрайты)
```

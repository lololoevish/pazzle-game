# План миграции Adventure Puzzle Game на GameMaker 2

## 📋 Обзор

Этот документ описывает пошаговый план перехода игры с Rust (macroquad) на GameMaker 2 (GML).

**Цель**: Создать полноценную версию игры в GameMaker 2 с сохранением всей функциональности Rust-версии.

**Приоритеты**:
1. Сохранение игровой механики
2. Перенос ассетов (спрайты, звуки)
3. Адаптация UI/UX под GameMaker
4. Оптимизация под платформы (Windows, Web, Mobile)

## 📝 Примечания

### Преимущества GameMaker 2
- Встроенные инструменты для UI и анимаций
- Легче для новичков
- Быстрая сборка
- Кроссплатформенность (Windows, Web, Mobile)
- Визуальный редактор

### Недостатки
- Больший размер exe
- Ограничения GML (по сравнению с Rust)
- Меньше контроля над низкоуровневыми операциями

### Рекомендации
- Использовать объектно-ориентированный подход в GML
- Хранить логику в скриптах, а не в объектах
- Использовать встроенные инструменты GameMaker
- Оптимизировать ассеты (сжатие спрайтов, аудио)

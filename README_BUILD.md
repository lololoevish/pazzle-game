# Инструкция по сборке игры в EXE

## Быстрый старт

### Способ 1: Автоматическая сборка (рекомендуется)
Просто запустите файл:
```
build_exe.bat
```

### Способ 2: Ручная сборка
```bash
# 1. Установите зависимости
pip install -r requirements.txt

# 2. Соберите exe
pyinstaller AdventurePuzzleGame.spec
```

## Результат
После успешной сборки исполняемый файл будет находиться в:
```
dist/AdventurePuzzleGame.exe
```

## Запуск игры
Просто запустите `AdventurePuzzleGame.exe` из папки `dist/`

## Управление в игре
- **ESC** - выход из уровня/возврат назад
- **Стрелки/WASD** - движение
- **Мышь** - взаимодействие с объектами

## Система сохранений
Игра автоматически сохраняет прогресс в памяти во время сеанса.
Для постоянных сохранений можно добавить систему файлов сохранений.

## Структура проекта
```
├── dist/                          # Готовый exe файл
│   └── AdventurePuzzleGame.exe
├── src/                           # Исходный код
│   ├── main.py                    # Точка входа
│   ├── scenes/                    # Игровые сцены
│   ├── entities/                  # Игровые объекты
│   └── utils/                     # Утилиты
├── AdventurePuzzleGame.spec       # Конфигурация PyInstaller
├── build_exe.bat                  # Скрипт сборки
└── requirements.txt               # Зависимости
```

## Возможные проблемы

### Ошибка: "pygame not found"
```bash
pip install pygame==2.5.2
```

### Ошибка: "pyinstaller not found"
```bash
pip install pyinstaller==6.5.0
```

### Большой размер exe
Это нормально для PyInstaller - он упаковывает Python и все библиотеки.
Размер обычно 30-50 МБ.

## Добавление иконки
1. Создайте файл `icon.ico` в корне проекта
2. В файле `AdventurePuzzleGame.spec` измените строку:
   ```python
   icon=None,  # на
   icon='icon.ico',
   ```

## Распространение
Можно распространять только папку `dist/` с файлом `AdventurePuzzleGame.exe`.
Все необходимое уже упаковано внутри.

## 🎯 Сценарии использования

### Сценарий 1: Первый запуск
1. 📖 Прочитать **`БЫСТРЫЙ_СТАРТ.txt`**
2. 🚀 Запустить **`ЗАПУСК_СБОРКИ.bat`**
3. 🎮 Играть!

### Сценарий 2: Начало разработки
1. 📖 Прочитать **`ДЛЯ_РАЗРАБОТЧИКА.md`**
2. 🔧 Установить зависимости: `pip install -r requirements.txt`
3. 💻 Запустить: `python src/main.py`
4. 📝 Изучить **`ПРИМЕРЫ_МОДИФИКАЦИЙ.md`**

### Сценарий 3: Добавление нового уровня
1. 📖 Открыть **`ДЛЯ_РАЗРАБОТЧИКА.md`** → "Как добавить новый уровень"
2. ✏️ Редактировать `src/scenes/gameplay.py`
3. ✏️ Обновить `src/main.py`
4. ✏️ Обновить `src/scenes/town.py`
5. 🧪 Тестировать: `python src/main.py`

### Сценарий 4: Добавление NPC
1. 📖 Открыть **`ПРИМЕРЫ_МОДИФИКАЦИЙ.md`** → Пример #6
2. ✏️ Редактировать `src/scenes/town.py`
3. 🧪 Тестировать: `python src/main.py`

### Сценарий 5: Изменение визуала
1. 📖 Открыть **`ПРИМЕРЫ_МОДИФИКАЦИЙ.md`** → Примеры #1-5
2. ✏️ Редактировать соответствующие файлы
3. 🧪 Тестировать: `python src/main.py`

### Сценарий 6: Сборка для распространения
1. 📖 Прочитать **`README_BUILD.md`**
2. 🔧 Проверить `AdventurePuzzleGame.spec`
3. 🚀 Запустить **`ЗАПУСК_СБОРКИ.bat`**
4. ✅ Протестировать `dist/AdventurePuzzleGame.exe`

### Сценарий 7: Решение проблем
1. 📖 Открыть **`СПРАВКА.md`** → "Решение проблем"
2. 📝 Проверить `game.log`
3. 🔍 Поискать в документации
4. 💡 Попробовать примеры из **`ПРИМЕРЫ_МОДИФИКАЦИЙ.md`**

## 🚀 Как использовать

### Для игроков

1. **Запустить сборку:**
   ```
   Двойной клик на: ЗАПУСК_СБОРКИ.bat
   ```

2. **Играть:**
   ```
   Запустить: dist\AdventurePuzzleGame.exe
   ```

3. **Прогресс сохраняется автоматически!**

### Для разработчиков

1. **Установить зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Разработка:**
   ```bash
   python src/main.py
   ```

3. **Сборка:**
   ```bash
   pyinstaller AdventurePuzzleGame.spec
   ```

4. **Модификация:**
   - См. `ПРИМЕРЫ_МОДИФИКАЦИЙ.md`
   - См. `ДЛЯ_РАЗРАБОТЧИКА.md`

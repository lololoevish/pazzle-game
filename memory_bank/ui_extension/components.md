# 🧩 Компоненты UI

## Обзор

Описывает переиспользуемые компоненты интерфейса.

## Кнопки

**Компонент**: `Button`

**Атрибуты**:
- `text` - текст кнопки
- `x`, `y` - позиция
- `width`, `height` - размеры
- `hovered` - наведен ли курсор
- `clicked` - нажата ли кнопка

**Методы**:
- `draw(screen)` - отрисовка
- `check_click(mouse_pos)` - проверка клика
- `update(mouse_pos)` - обновление состояния

## Текст

**Компонент**: `Text`

**Атрибуты**:
- `text` - текст
- `x`, `y` - позиция
- `font` - шрифт
- `color` - цвет

**Методы**:
- `draw(screen)` - отрисовка
- `set_text(text)` - изменение текста

## Прогресс-бар

**Компонент**: `ProgressBar`

**Атрибуты**:
- `x`, `y` - позиция
- `width`, `height` - размеры
- `value` - текущее значение
- `max_value` - максимальное значение

**Методы**:
- `draw(screen)` - отрисовка
- `set_value(value)` - изменение значения

## Портрет

**Компонент**: `Portrait`

**Атрибуты**:
- `image` - изображение
- `x`, `y` - позиция
- `opacity` - прозрачность

**Методы**:
- `draw(screen)` - отрисовка
- `set_opacity(opacity)` - изменение прозрачности

## Диалоговое окно

**Компонент**: `DialogueBox`

**Атрибуты**:
- `text` - текст
- `portrait` - портрет
- `x`, `y` - позиция
- `width`, `height` - размеры

**Методы**:
- `draw(screen)` - отрисовка
- `update()` - обновление анимации текста

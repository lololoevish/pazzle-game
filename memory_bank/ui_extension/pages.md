# 📄 Страницы UI

## Обзор

Описывает страницы интерфейса и их компоненты.

## Главное меню

**Страница**: `MenuScene`

**Компоненты**:
- Заголовок игры
- Кнопка "Играть"
- Кнопка "Настройки"
- Кнопка "Выход"
- Анимированный фон

**Поток данных**:
1. Пользователь выбирает опцию
2. Сцена обновляет состояние
3. При завершении перехода возвращается результат

## Город-хаб

**Страница**: `TownScene`

**Компоненты**:
- Список уровней
- Индикаторы прогресса
- NPC с мини-играми
- Кнопка "Назад"

**Поток данных**:
1. Пользователь выбирает уровень
2. Сцена обновляет состояние
3. При выборе уровня запускается GameplayScene

## Игровой процесс

**Страница**: `GameplayScene`

**Компоненты**:
- Игровое поле
- Индикаторы (время, очки)
- Кнопка "Выход"

**Поток данных**:
1. Пользователь взаимодействует с головоломкой
2. Сцена обновляет состояние
3. При победе возвращается в город

## Мини-игры

**Страница**: `MiniGameScene`

**Компоненты**:
- Игровое поле мини-игры
- Индикаторы (очки, попытки)
- Кнопка "Выход"

**Поток данных**:
1. Пользователь играет в мини-игру
2. Сцена обновляет состояние
3. При завершении возвращается в город с наградой

## Настройки

**Страница**: Планируется

**Компоненты**:
- Выбор громкости
- Управление
- Тема
- Кнопка "Назад"

**Поток данных**:
1. Пользователь меняет настройки
2. Сцена обновляет состояние
3. При выходе сохраняются настройки

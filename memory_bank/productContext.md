# Product Context

## Зачем нужен проект
Система Memory Bank необходима для обеспечения устойчивого контекста работы AI-агентов, которые не обладают долгосрочной памятью между сессиями.

## Проблемы пользователей
- Потеря контекста между сессиями работы с AI-агентами
- Отсутствие единого источника правды о состоянии проекта
- Сложность отслеживания прогресса выполнения задач
- Необходимость повторного объяснения архитектурных решений

## Целевая аудитория
Разработчики и AI-агенты, работающие над долгосрочными проектами с необходимостью поддержания контекста.

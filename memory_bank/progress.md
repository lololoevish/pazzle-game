## Зависимости

- Требуется GameMaker Studio 2 (версия 2022.6 или новее) для разработки и ручной runtime-проверки GMS2-версии.
- Для альтернативного запуска без GameMaker требуется Bun и браузер; команды выполняются из `web-port/`.
- Rust-ветка остаётся полезной как эталон исторического маршрута и части игровых решений.
- Локальные ассеты и конфигурации проекта используются как база для дальнейшей полировки GMS2-версии.

# Tech Context

## Стек технологий
- **Документация**: Markdown
- **Контроль версий**: Git
- **Пакетный менеджер**: bun (согласно AGENTS.md)
- **Линтинг**: biome (для кода, не для .md файлов)

## Окружение
- **ОС**: Windows (C:\Users\admin)
- **Рабочая директория**: C:\Users\admin
- **Git репозиторий**: инициализирован в C:\Users\admin

## Ограничения
- Markdown-файлы не проверяются через Biome
- Сервер разработки управляется пользователем, не агентом
- Все изменения должны быть зафиксированы в Git

## CI/CD
На данный момент CI/CD не настроен. Требуется ручной коммит и пуш изменений.

# 🎬 Сцены игры

## Обзор

Сцены управляют состояниями игры и отвечают за обработку ввода, обновление состояния и отрисовку.

## Список сцен

### MenuScene

**Файл**: `src/scenes/menu.py` / `src_rust/scenes/menu.rs`

**Назначение**: Главное меню игры

**Функции**:
- Отображение заголовка и опций
- Анимация фона
- Обработка выбора пользователя

**Опции**:
- Играть
- Настройки
- Выход

### TownScene

**Файл**: `src/scenes/town.py` / `src_rust/scenes/town.rs`

**Назначение**: Город-хаб для выбора уровней

**Функции**:
- Отображение списка уровней
- Индикация прогресса
- Взаимодействие с NPC

**Уровни**:
- Уровень 1: Лабиринт
- Уровень 2: Поиск слов
- Уровень 3: Память
- Уровни 4-6: Планируется

### GameplayScene

**Файл**: `src/scenes/gameplay.py` / `src_rust/scenes/gameplay.rs`

**Назначение**: Игровой процесс

**Функции**:
- Загрузка головоломки
- Обработка ввода
- Проверка победы

### MiniGameScene

**Файл**: `src/scenes/minigames.py`

**Назначение**: Мини-игры с NPC

**Мини-игры**:
- Угадай число
- Камень-ножницы-бумага
- Викторина

## Паттерн State

Все сцены реализуют паттерн State:

```python
class Scene:
    def handle_events(self, events): ...
    def update(self, keys): ...
    def draw(self, screen): ...
    def is_transition_complete(self): ...
    def get_result(self): ...
```

## Смена сцен

Сцены меняются через проверку `is_transition_complete()` и получение результата через `get_result()`.

# 🗃️ Legacy код

## Обзор

Этот раздел содержит информацию о устаревшем коде и его миграции.

## Python версия

**Статус**: Устаревает

**Причина**: Rust версия имеет лучшую производительность и меньший размер.

**Что делать**:
- Используйте Rust версию для продакшена
- Python версию можно оставить для обучения и модификаций

## Старые файлы

**Файлы для удаления**:
- `src/main.py` - заменён на `src_rust/main.rs`
- `src/scenes/*.py` - заменены на `src_rust/scenes/*.rs`
- `src/entities/*.py` - заменены на `src_rust/entities/*.rs`
- `src/utils/*.py` - заменены на `src_rust/utils/*.rs`

**Файлы для сохранения** (для обучения):
- `src/main.py` - можно оставить как пример Python
- `src/scenes/*.py` - можно оставить как пример Python
- `src/entities/*.py` - можно оставить как пример Python
- `src/utils/*.py` - можно оставить как пример Python

## Миграция

### Сценарий 1: Полный переход на Rust

1. Удалить все Python файлы
2. Обновить README
3. Удалить requirements.txt
4. Обновить документацию

### Сценарий 2: Параллельное использование

1. Оставить Python файлы
2. Обновить README с описанием обеих версий
3. Обновить документацию

## Известные проблемы

1. Антивирусы могут ложно срабатывать на exe (добавьте в исключения)
2. Размер exe ~50-80 МБ (это нормально для PyInstaller)
3. Зависимость от Python (для разработки)

## Рекомендации

1. Используйте Rust версию для продакшена
2. Оставьте Python версию для обучения и модификаций
3. Обновляйте документацию при изменениях

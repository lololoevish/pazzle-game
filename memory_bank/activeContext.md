## 🎮 Активные решения

### Архитектурные решения

1. **State Pattern** для управления сценами
2. **Trait Objects** для гибкости в Rust
3. **JSON** для сохранений (обе версии)
4. **Модульная структура** для расширения

### Технические решения

1. **Python**: Pygame 2.5.2 + PyInstaller
2. **Rust**: macroquad 0.4 + serde + serde_json
3. **Сборка**: Автоматические скрипты (.bat)

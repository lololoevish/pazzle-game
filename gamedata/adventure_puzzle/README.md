# Проект GameMaker Studio 2 - Adventure Puzzle Game

Это GameMaker Studio 2 проект для игры "Adventure Puzzle Game" - приключенческой 2D игры с головоломками.

## Структура проекта

### Основные папки:
- `Objects` - игровые объекты
- `Rooms` - уровни и экраны
- `Sprites` - графические ресурсы
- `Sounds` - аудиофайлы
- `Scripts` - игровые функции
- `Shaders` - эффекты рендеринга
- `Fonts` - шрифты
- `Timelines` - анимации по времени
- `Paths` - пути движения
- `Configs` - конфигурации сборки

## Архитектура

Проект следует архитектурному паттерну, описанному в `docs/gms_architecture.md`:
- Централизованное управление через `obj_game_manager`
- Система состояний игры
- Модульная структура объектов
- Единая система сохранений

## Требования

- GameMaker Studio 2 (версия 2022.6 или новее)
- Windows 10 или новее для разработки
- DirectX 11 для запуска

## Сборка проекта

1. Откройте проект через `adventure_puzzle.yyp`
2. Выберите нужную конфигурацию сборки
3. Используйте Build & Run для тестирования
4. Используйте Create Package для создания релизной версии

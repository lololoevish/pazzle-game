# 🎮 Adventure Puzzle Game - Инструкция

## 🚀 Быстрый старт

### Способ 1: Автоматическая сборка (РЕКОМЕНДУЕТСЯ)
Просто запустите файл:
```
ЗАПУСК_СБОРКИ.bat
```
Скрипт автоматически:
- Установит все зависимости
- Соберёт .exe файл
- Предложит запустить игру

### Способ 2: Ручная сборка
```bash
pip install -r requirements.txt
pyinstaller AdventurePuzzleGame.spec
```

## 📁 Где найти готовую игру?
После сборки исполняемый файл будет здесь:
```
dist/AdventurePuzzleGame.exe
```

## 🎯 Управление в игре

### Меню
- **↑↓** или **W/S** - выбор пункта меню
- **Enter** или **Пробел** - подтверждение

### В игре
- **Стрелки** или **WASD** - движение персонажа
- **Мышь** - взаимодействие с объектами (переключатели, NPC)
- **ESC** - выход из уровня/возврат в город/меню

## 💾 Система сохранений

Игра **автоматически сохраняет** прогресс при выходе в файл `savegame.json` рядом с exe.

Сохраняется:
- Пройденные уровни
- Золото и предметы
- Выигранные мини-игры

При следующем запуске прогресс загрузится автоматически!

## 🏗️ Структура игры

```
├── dist/
│   ├── AdventurePuzzleGame.exe    ← Готовая игра
│   └── savegame.json              ← Файл сохранения (создаётся автоматически)
├── src/                           ← Исходный код
│   ├── main.py                    ← Точка входа
│   ├── scenes/                    ← Игровые сцены
│   │   ├── menu.py               ← Главное меню
│   │   ├── town.py               ← Город (хаб)
│   │   ├── gameplay.py           ← Игровой процесс
│   │   └── minigames.py          ← Мини-игры с NPC
│   ├── entities/                  ← Игровые объекты
│   │   ├── player.py             ← Игрок
│   │   └── item.py               ← Предметы
│   └── utils/                     ← Утилиты
│       ├── helpers.py            ← Вспомогательные функции
│       ├── visual_effects.py     ← Визуальные эффекты
│       └── save_system.py        ← Система сохранений
├── AdventurePuzzleGame.spec       ← Конфигурация PyInstaller
├── ЗАПУСК_СБОРКИ.bat             ← Автоматическая сборка
└── requirements.txt               ← Зависимости Python
```

## 🎨 Особенности игры

- **6 уровней** с головоломками
- **Город-хаб** для выбора уровней
- **NPC с мини-играми** (угадай число, камень-ножницы-бумага, викторина)
- **Система инвентаря** (золото, предметы)
- **Автосохранение** прогресса
- **Красивое анимированное меню**

## 🔧 Возможные проблемы

### "pygame not found"
```bash
pip install pygame==2.5.2
```

### "pyinstaller not found"
```bash
pip install pyinstaller==6.5.0
```

### Большой размер exe (30-50 МБ)
Это нормально! PyInstaller упаковывает Python и все библиотеки внутрь exe.

### Антивирус блокирует exe
Это ложное срабатывание. PyInstaller-приложения иногда вызывают подозрения.
Добавьте файл в исключения антивируса.

## 📦 Распространение игры

Можно распространять только папку `dist/` с файлом `AdventurePuzzleGame.exe`.
Всё необходимое уже упаковано внутри!

Пользователям НЕ нужно устанавливать Python или библиотеки.

## 🎯 Добавление иконки

1. Создайте файл `icon.ico` в корне проекта
2. Откройте `AdventurePuzzleGame.spec`
3. Найдите строку `icon=None,`
4. Замените на `icon='icon.ico',`
5. Пересоберите проект

## 🛠️ Разработка

Для запуска из исходников (без сборки):
```bash
python src/main.py
```

Для тестирования изменений не нужно каждый раз собирать exe!

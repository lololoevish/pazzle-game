# Спрайты игры

## Структура папки

```
assets/sprites/
├── player_down.png    - Спрайт игрока (смотрит вниз)
├── player_up.png      - Спрайт игрока (смотрит вверх)
├── player_left.png    - Спрайт игрока (смотрит влево)
├── player_right.png   - Спрайт игрока (смотрит вправо)
├── npc_roan.png       - Спрайт механика Роана
├── npc_tellah.png     - Спрайт архивариуса Теля
├── npc_yore.png       - Спрайт старосты Иара
├── item.png           - Спрайт предмета
├── enemy.png          - Спрайт врага
├── lever.png          - Спрайт рычага
├── platform.png       - Спрайт платформы
├── clockwork_emblem.png - Декор пещеры 3 (часы)
├── mirror_shard.png   - Декор пещеры 4 (зеркала)
├── crystal_cluster.png - Декор пещеры 5 (кристаллы)
├── core_spire.png     - Декор пещеры 6 (ядро)
└── README.md          - Этот файл
```

## Формат спрайтов

- **Формат:** PNG с прозрачным фоном
- **Размер:** 64x64 пикселей
- **Цветовая глубина:** 32 бита (RGBA)
- **Стиль:** Deltarune (детальные пиксельные спрайты с выразительными глазами и контурами)

## Как заменить спрайты

1. Создай свои PNG-файлы с нужными размерами
2. Замени файлы в папке `assets/sprites/`
3. Пересобери игру: `cargo build --release`

## Примечания

- Спрайты игрока и NPC созданы в стиле Deltarune с детализированными глазами, контурами и выразительными чертами
- Для анимации можно создать несколько кадров и менять их по таймеру
- Все пути задаются относительно корня проекта

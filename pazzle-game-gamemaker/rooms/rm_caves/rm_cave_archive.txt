# Комнаты GameMaker - Пещера: Архив

## rm_cave_archive
- Размер: 800x600
- Объекты:
  - obj_game_manager (0, 0)
  - obj_puzzle (логика головоломки)
  - сетка букв/слова (пользовательские объекты)
  - obj_lever (выход)
- Слои:
  - Background: библиотечная/архивная тема
  - LettersGrid: сетка букв
  - Selection: выбранные буквы
  - GUI: интерфейс выбора слов
  - Instances: интерактивные элементы

## Описание:
Вторая пещера с головоломкой "Поиск слов".

## Логика:
- Сетка букв 10x10 (или другая)
- Выбор букв по горизонтали/вертикали/диагонали
- Поиск слов из заранее определенного списка
- Подсветка найденных слов
- Потянуть рычаг при нахождении всех слов

## Состояния:
- global.current_state = "PLAYING"
- puzzle_level = 2

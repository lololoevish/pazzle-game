# Комнаты GameMaker - Пещера: Платформер

## rm_cave_platformer
- Размер: 800x600 (может быть больше с прокруткой)
- Объекты:
  - obj_game_manager (0, 0)
  - obj_player (начальная позиция)
  - платформы
  - объекты прыжков/движения
  - коллекционные элементы
  - obj_lever (выход)
- Слои:
  - Background: пещерный пейзаж
  - Platforms: платформы
  - Collectibles: собираемые элементы
  - Enemies: враги (опционально)
  - Instances: игрок

## Описание:
Пятая пещера с головоломкой-платформером.

## Логика:
- Движение и прыжки игрока
- Преодоление препятствий
- Сбор элементов (кристаллов)
- Коллизии с платформами
- Потянуть рычаг при сборе всех элементов

## Состояния:
- global.current_state = "PLAYING"
- puzzle_level = 5

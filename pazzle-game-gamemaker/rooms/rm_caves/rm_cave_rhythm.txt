# Комнаты GameMaker - Пещера: Ритм

## rm_cave_rhythm
- Размер: 800x600
- Объекты:
  - obj_game_manager (0, 0)
  - obj_puzzle (логика головоломки)
  - кнопки/элементы паттерна (4 элемента)
  - obj_lever (выход)
- Слои:
  - Background: тема с ритмом/музыкой
  - Buttons: кнопки паттерна
  - Effects: анимации при нажатиях
  - GUI: счетчик раундов

## Описание:
Третья пещера с головоломкой "Паттерн-память" (Simon Says).

## Логика:
- Показывается последовательность светящихся кнопок
- Игрок должен повторить эту последовательность
- Каждый раунд увеличивает длину паттерна
- При ошибке - сброс или повтор уровня
- Потянуть рычаг при прохождении

## Состояния:
- global.current_state = "PLAYING"
- puzzle_level = 3

# Проект Adventure Puzzle Game на GameMaker 2

## Обзор

Это GameMaker 2 версия проекта Adventure Puzzle Game, мигрированная из Rust (macroquad). Проект включает в себя 6 уровней с головоломками, систему NPC с мини-играми, систему прогресса и сохранений.

## Структура проекта

```
pazzle-game-gamemaker/
├── objects/                  # Объекты игры
│   ├── obj_game_manager      # Управление состоянием игры

│   ├── scr_ui_manager       # UI элементы
│   └── puzzle_scripts/      # Скрипты головоломок
├── rooms/                   # Игровые комнаты
│   ├── rm_menu              # Главное меню
│   ├── rm_town              # Город-хаб
```

## Основные компоненты

### Объекты

- `obj_game_manager`: Центральный объект управления состоянием игры и переходами между комнатами
- `obj_player`: Объект игрока с управлением и физикой
- `obj_puzzle`: Базовый класс для всех головоломок
- `obj_lever`: Объект рычага, сигнализирующий о завершении уровня
- `obj_npc`: Объекты для взаимодействия с персонажами
- `obj_interactable`: Базовый класс для всех интерактивных элементов

### Скрипты

- `scr_game_state`: Управление прогрессом игры и текущим состоянием
- `scr_save_system`: Система сохранения и загрузки прогресса
- `scr_audio_manager`: Управление звуками и музыкой
- `scr_ui_manager`: Интерфейс пользователя
- `scr_puzzle_manager`: Менеджер для управления различными типами головоломок

### Типы головоломок

1. **Лабиринт (scr_maze_puzzle)**: Двигайтесь от начала до конца лабиринта
2. **Поиск слов (scr_word_search_puzzle)**: Найдите все слова в сетке
3. **Ритм (scr_rhythm_puzzle)**: Повторите последовательность паттернов
4. **Пары (scr_memory_match_puzzle)**: Найдите совпадающие пары карточек
5. **Платформер (scr_platformer_puzzle)**: Прыгайте по платформам и собирайте кристаллы
6. **Финальная (scr_final_challenge_puzzle)**: Сложная головоломка с препятствиями и ограничением по времени

## Системы

### Система прогресса
- Уровни разблокируются по цепочке через опускание рычага
- Повторный вход в уже пройденный уровень позволяет сразу опустить рычаг без решения головоломки
- Система наград за мини-игры с NPC

### Система сохранений
- Использует JSON для хранения прогресса
- Сохраняет состояние уровней, золото, предметы и прогресс NPC-мини-игр
- Поддерживает создание новой игры и загрузку существующего прогресса

### Аудио-система
- Поддержка музыкальных тем для разных сцен
- Звуковые эффекты для UI и игровых событий
- Управление громкостью и режимом отключения звука

## Тестирование

- `UNIT_TESTS.md`: Сценарии модульного тестирования

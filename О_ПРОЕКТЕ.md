# 📖 О проекте Adventure Puzzle Game

## 🎯 Концепция

Adventure Puzzle Game - это 2D приключенческая игра с головоломками, созданная на Python с использованием библиотеки Pygame. Игра демонстрирует полный цикл разработки от исходного кода до готового исполняемого файла.

## 🏗️ Архитектура

### Модульная структура

Проект построен по модульному принципу, где каждый компонент отвечает за свою область:

```
Игровой движок (main.py)
    ↓
Сцены (scenes/)
    ├── Меню (menu.py)
    ├── Город (town.py)
    ├── Геймплей (gameplay.py)
    └── Мини-игры (minigames.py)
    ↓
Сущности (entities/)
    ├── Игрок (player.py)
    └── Предметы (item.py)
    ↓
Утилиты (utils/)
    ├── Помощники (helpers.py)
    ├── Эффекты (visual_effects.py)
    └── Сохранения (save_system.py)
```

### Паттерны проектирования

1. **State Pattern** - Управление состояниями игры (меню, город, игра, мини-игры)
2. **Component Pattern** - Разделение логики на независимые компоненты
3. **Observer Pattern** - Система событий и переходов между сценами
4. **Singleton Pattern** - Система сохранений

## 🎮 Игровая механика

### Основной геймплей

- **Головоломки с переключателями**: Игрок должен активировать переключатели в правильном порядке
- **Движущиеся препятствия**: Динамические элементы, усложняющие прохождение
- **Система прогресса**: Отслеживание пройденных уровней

### Город-хаб

- Центральная локация для выбора уровней
- NPC с мини-играми
- Визуальная индикация доступных уровней

### Мини-игры

1. **Угадай число** - Логическая игра с подсказками
2. **Камень-ножницы-бумага** - Классическая игра на удачу
3. **Викторина** - Проверка знаний

### Система прогрессии

- Золото за победы в мини-играх
- Предметы-награды
- Сохранение прогресса между сессиями

## 💾 Техническая реализация

### Система сохранений

```python
Формат: JSON
Расположение: Рядом с exe или в корне проекта
Содержимое:
    - Прогресс по уровням
    - Инвентарь игрока
    - Выигранные мини-игры
```

### Оптимизация производительности

- Ограничение FPS (60)
- Эффективная отрисовка (только видимые объекты)
- Кэширование поверхностей
- Оптимизация коллизий

### Сборка в exe

Используется PyInstaller с настроенным spec-файлом:
- Упаковка всех зависимостей
- Включение модулей игры
- Оптимизация размера (UPX)
- Поддержка ресурсов

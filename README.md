# Adventure Puzzle Game

Приключенческая 2D puzzle/adventure игра с хабом, 12 пещерами, NPC-активностями, системой сохранений, аудио-слоем и элементами визуальной новеллы.

## Текущий Статус

Основная продуктовая версия проекта находится в `pazzle-game-gamemaker/` и разрабатывается под GameMaker Studio 2.

Rust-ветка сохраняется как исторический референс маршрута, UX и ранних механик, но больше не является основной исполняемой целью проекта.

## Быстрый Старт

1. Откройте GameMaker Studio 2.
2. Откройте проект из папки `pazzle-game-gamemaker/`.
3. Если GameMaker просит выбрать файл проекта, используйте `.yyp` внутри этой папки.

Не открывайте корень репозитория как GameMaker-проект.

## Основной Маршрут

`Menu -> Town -> Cave 1..12 -> Victory -> Town`

Ключевые системы:

- прогресс 12 пещер;
- хаб с тремя NPC;
- сохранения и восстановление состояния;
- рычаги и последовательное открытие уровней;
- аудио, SFX и контекстная музыка;
- VN-диалоги и портретная система;
- тестовые сценарии для основных runtime-модулей.

## Проверки

`.gml` файлы сейчас не обрабатываются Biome текущей конфигурацией. Основная проверка качества выполняется через ревизию кода, GML test scenarios и ручной runtime-проход в GameMaker Studio 2.

# 🎮 Adventure Puzzle Game

Приключенческая игра с головоломками на Python + Pygame

> 📖 **Новичок?** Начните с файла **[БЫСТРЫЙ_СТАРТ.txt](БЫСТРЫЙ_СТАРТ.txt)**  
> 🗺️ **Ищете что-то конкретное?** Смотрите **[НАВИГАЦИЯ.md](НАВИГАЦИЯ.md)**

## ✨ Особенности

- 🎯 **6 уровней** с уникальными головоломками
- 🏘️ **Город-хаб** для выбора уровней
- 🎲 **NPC с мини-играми** (угадай число, камень-ножницы-бумага, викторина)
- 💰 **Система инвентаря** (золото, предметы, награды)
- 💾 **Автосохранение** прогресса
- 🎨 **Красивое анимированное меню** с эффектами
- 🎵 **Готово к добавлению** звуков и музыки
- 📦 **Сборка в .exe** - одним кликом!

## 🚀 Быстрый старт

### Для игроков

1. Запустите `ЗАПУСК_СБОРКИ.bat`
2. Дождитесь окончания сборки
3. Играйте! Файл будет в `dist/AdventurePuzzleGame.exe`

### Для разработчиков

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск из исходников (для разработки)
python src/main.py

# Сборка в exe
pyinstaller AdventurePuzzleGame.spec
```

## 🎮 Управление

### Меню
- **↑↓** или **W/S** - выбор
- **Enter** - подтверждение

### В игре
- **Стрелки** или **WASD** - движение
- **Мышь** - взаимодействие
- **ESC** - назад/выход

## 💾 Система сохранений

Игра автоматически сохраняет:
- Пройденные уровни
- Золото и предметы
- Выигранные мини-игры

Файл сохранения: `savegame.json` (создаётся автоматически)

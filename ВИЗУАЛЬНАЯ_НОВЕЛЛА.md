# 🎭 Визуальная новелла - Руководство

## 📖 Что добавлено

В игру добавлена полноценная система визуальной новеллы с:
- Диалоговыми окнами
- Портретами персонажей
- Анимацией текста
- Сюжетными сценами

## 🚀 Быстрый старт

### Простой диалог

```python
from utils.dialogue_system import DialogueSequence

# Создать систему диалогов
dialogue = DialogueSequence(800, 600)

# Запустить диалог
dialogues = [
    ("Герой", "Привет!"),
    ("NPC", "Здравствуй, путник!"),
]
dialogue.start_sequence(dialogues)

# В игровом цикле
dialogue.handle_events(events)
dialogue.update()
dialogue.draw(screen)
```

### С портретами персонажей

```python
from utils.dialogue_system import VisualNovelScene

# Создать сцену
vn = VisualNovelScene(800, 600)

# Добавить персонажей
vn.add_character("Герой", position='left', color=(120, 80, 60))
vn.add_character("Мудрец", position='right', color=(80, 60, 120))

# Показать персонажей
vn.show_character("Герой")
vn.show_character("Мудрец")

# Запустить диалог
dialogues = [
    ("Герой", "Что мне делать?"),
    ("Мудрец", "Пройди все испытания!"),

vn.start_dialogue(dialogues)
```

## 🎨 Компоненты системы

### 1. DialogueBox
Базовое диалоговое окно с анимацией текста.

**Возможности:**
- Анимация появления текста
- Имя говорящего
- Автоматический перенос строк
- Индикатор продолжения

**Управление:**
- Enter/Пробел - пропустить анимацию или продолжить

### 2. DialogueSequence
Последовательность диалогов.

**Возможности:**
- Цепочка диалогов
- Автоматический переход
- Отслеживание завершения

### 3. CharacterPortrait
Портрет персонажа.

**Возможности:**
- Анимация появления/исчезновения
- Настраиваемый цвет
- Позиционирование (left/center/right)

### 4. VisualNovelScene
Полноценная сцена визуальной новеллы.

**Возможности:**
- Управление портретами
- Диалоги
- Фон
- Полная интеграция

## 💡 Советы

1. **Длина текста**: Держите диалоги короткими (2-3 строки)
2. **Имена**: Используйте короткие имена персонажей
3. **Цвета**: Выбирайте контрастные цвета для портретов
4. **Позиции**: left для главного героя, right для NPC
5. **Анимация**: Дайте игроку возможность пропустить её

## 🐛 Решение проблем

### Текст не помещается
Уменьшите `self.box_height` или разбейте на несколько диалогов.

### Портреты перекрываются
Используйте разные позиции: 'left', 'center', 'right'.

### Анимация слишком быстрая/медленная
Измените `self.text_speed` в DialogueBox.

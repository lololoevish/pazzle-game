# 🎮 Adventure Puzzle Game - Справка

## 🚀 Быстрые команды

### Windows (Batch файлы)
```batch
КОМАНДЫ.bat              # Интерактивное меню всех команд
ЗАПУСК_СБОРКИ.bat       # Собрать и запустить игру
build_exe.bat            # Только сборка
очистка.bat              # Очистить временные файлы
```

### Командная строка
```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск из исходников
python src/main.py

# Сборка в exe
pyinstaller AdventurePuzzleGame.spec

# Очистка
rmdir /s /q build dist
```

- `dist/AdventurePuzzleGame.exe` - Готовая игра

```
│   ├── player.py       # Игрок

└── utils/               # Утилиты
    ├── helpers.py      # Помощники
    ├── visual_effects.py # Эффекты
    └── save_system.py  # Сохранения
```

### Добавление контента

#### Новый уровень
1. Открыть `src/scenes/gameplay.py`
2. Добавить в метод `load_level()`
3. Обновить `game_progress` в `src/main.py`
4. Добавить дверь в `src/scenes/town.py`

#### Новый NPC
1. Открыть `src/scenes/town.py`
2. Добавить в список `self.npcs`
3. Указать тип мини-игры и награду

#### Новая мини-игра
1. Открыть `src/scenes/minigames.py`
2. Добавить методы setup/update/draw
3. Добавить обработку в `__init__()`

## 💾 Система сохранений

### Формат файла (savegame.json)
```json
{
  "progress": {
    "1": {"completed": true, "lever_pulled": false},
    "2": {"completed": false, "lever_pulled": false}
  },
  "inventory": {
    "gold": 150,
    "items": ["Золотой ключ"],
    "won_minigames": ["Торговец"]
  }
}
```

### Расположение
- Из исходников: `./savegame.json`
- Из exe: `savegame.json` рядом с exe

### API
```python
from utils.save_system import save_game, load_game, has_save, delete_save

# Проверить наличие
if has_save():
    print("Сохранение найдено")

# Загрузить
data = load_game()
if data:
    progress, inventory = data

# Сохранить
save_game(game_progress, player_inventory)

# Удалить
delete_save()
```

## 🐛 Решение проблем

### Ошибка: "pygame not found"
```bash
pip install pygame==2.5.2
```

### Ошибка: "pyinstaller not found"
```bash
pip install pyinstaller==6.5.0
```

### Ошибка: "No module named 'src'"
Запускайте из корня проекта:
```bash
python src/main.py  # Правильно
cd src && python main.py  # Неправильно
```

### Антивирус блокирует exe
Добавьте в исключения - это ложное срабатывание

### Большой размер exe
Это нормально для PyInstaller (~30-50 MB)

### Сохранения не работают
Проверьте права на запись в папку с exe

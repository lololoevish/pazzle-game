# Система достижений (Achievements)

## Обзор

Система достижений добавляет дополнительную мотивацию для игроков, награждая их за выполнение различных задач и целей в игре.

## Структура достижений

### Категории

1. **Прогресс** - Завершение уровней и основного сюжета
2. **Мастерство** - Выполнение задач с особыми условиями
3. **Исследование** - Взаимодействие с NPC и миром
4. **Коллекционирование** - Сбор предметов и ресурсов
5. **Секретные** - Скрытые достижения

## Список достижений

### Прогресс (Progress)

#### ACH_FIRST_STEPS
- **Название**: Первые шаги
- **Описание**: Завершите первую пещеру
- **Условие**: Опустить рычаг в уровне 1
- **Награда**: 10 золота
- **Иконка**: spr_ach_first_steps

#### ACH_HALFWAY_THERE
- **Название**: На полпути
- **Описание**: Завершите первые 6 пещер
- **Условие**: Опустить рычаги в уровнях 1-6
- **Награда**: 50 золота
- **Иконка**: spr_ach_halfway

#### ACH_EXPEDITION_COMPLETE
- **Название**: Экспедиция завершена
- **Описание**: Завершите все 12 пещер
- **Условие**: Опустить рычаги во всех уровнях
- **Награда**: 100 золота, специальный предмет
- **Иконка**: spr_ach_expedition_complete

#### ACH_PERFECT_RUN
- **Название**: Безупречное прохождение
- **Описание**: Завершите все пещеры без единой смерти
- **Условие**: global.death_count == 0 && все уровни пройдены
- **Награда**: 200 золота
- **Иконка**: spr_ach_perfect_run

### Мастерство (Mastery)

#### ACH_SPEED_DEMON
- **Название**: Демон скорости
- **Описание**: Завершите любую пещеру менее чем за 2 минуты
- **Условие**: Время прохождения < 120 секунд
- **Награда**: 30 золота
- **Иконка**: spr_ach_speed_demon

#### ACH_PUZZLE_MASTER
- **Название**: Мастер головоломок
- **Описание**: Решите головоломку с первой попытки в 5 разных пещерах
- **Условие**: Счетчик perfect_puzzles >= 5
- **Награда**: 50 золота
- **Иконка**: spr_ach_puzzle_master

#### ACH_NO_HINTS
- **Название**: Без подсказок
- **Описание**: Завершите 3 пещеры без использования подсказок
- **Условие**: hints_used == 0 в 3 уровнях
- **Награда**: 40 золота
- **Иконка**: spr_ach_no_hints

### Исследование (Exploration)

#### ACH_SOCIAL_BUTTERFLY
- **Название**: Душа компании
- **Описание**: Поговорите со всеми NPC в городе
- **Условие**: Диалог с Elder, Mechanic, Archivist
- **Награда**: 20 золота
- **Иконка**: spr_ach_social

#### ACH_TRIAL_MASTER
- **Название**: Мастер испытаний
- **Описание**: Завершите все мини-игры NPC
- **Условие**: elder_trial + mechanic_training + archivist_quiz
- **Награда**: 60 золота
- **Иконка**: spr_ach_trial_master

#### ACH_LORE_KEEPER
- **Название**: Хранитель знаний
- **Описание**: Прочитайте все таблички и записи в пещерах
- **Условие**: Счетчик lore_items_read == total_lore_items
- **Награда**: 30 золота
- **Иконка**: spr_ach_lore_keeper

### Коллекционирование (Collection)

#### ACH_TREASURE_HUNTER
- **Название**: Охотник за сокровищами
- **Описание**: Соберите 100 золота
- **Условие**: global.game_progress.gold >= 100
- **Награда**: Специальный предмет
- **Иконка**: spr_ach_treasure_hunter

#### ACH_CRYSTAL_COLLECTOR
- **Название**: Коллекционер кристаллов
- **Описание**: Соберите все кристаллы в пещерах
- **Условие**: Счетчик crystals_collected == total_crystals
- **Награда**: 50 золота
- **Иконка**: spr_ach_crystal_collector

#### ACH_ITEM_HOARDER
- **Название**: Накопитель
- **Описание**: Соберите 10 различных предметов
- **Условие**: array_length(global.game_progress.items) >= 10
- **Награда**: 40 золота
- **Иконка**: spr_ach_item_hoarder

## Технические детали

### Формат данных

```json
{
  "achievement_id": "ACH_FIRST_STEPS",
  "name": "Первые шаги",
  "description": "Завершите первую пещеру",
  "category": "progress",
  "unlocked": false,
  "unlock_date": null,
  "progress": 0,
  "progress_max": 1,
  "reward_gold": 10,
  "reward_items": [],
  "icon": "spr_ach_first_steps",
  "hidden": false
}
```

### Условия разблокировки

Каждое достижение имеет функцию проверки условия:

```gml
function check_achievement_condition(achievement_id) {
    switch (achievement_id) {
        case "ACH_FIRST_STEPS":
            return global.game_progress.levels[0].lever_pulled;
        
        case "ACH_HALFWAY_THERE":
            var count = 0;
            for (var i = 0; i < 6; i++) {
                if (global.game_progress.levels[i].lever_pulled) count++;
            }
            return count >= 6;
        
        // ... другие условия
    }
    return false;
}
```

### Уведомления

При разблокировке достижения показывается уведомление:
- Иконка достижения
- Название
- Описание
- Награда
- Звуковой эффект

### Сохранение

Достижения сохраняются в отдельном файле:
- `achievements_save.json`
- Независимо от игрового прогресса
- Сохраняются даже при сбросе игры

## Интеграция

### Проверка достижений

Достижения проверяются в ключевых точках:
- После завершения уровня
- После взаимодействия с NPC
- После сбора предметов
- В конце каждого фрейма (для счетчиков)

### UI

Экран достижений доступен из:
- Главное меню
- Пауза в игре
- Город-хаб (специальная доска)

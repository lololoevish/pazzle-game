# Тематические ассеты пещер 3-6

Этот документ описывает тематические ассеты для пещер 3-6 в GameMaker 2 версии игры.

## Обзор

Каждая пещера имеет уникальную тематику, цветовую палитру и набор декоративных элементов, которые создают атмосферу и визуально отличают уровни друг от друга.

## Пещера 3: Часовой механизм (Rhythm)

**Тема**: `clockwork_mechanism`  
**Головоломка**: Ритм-паттерны

### Цветовая палитра
- Primary: `#2c1810` (темно-коричневый)
- Accent: `#c9a961` (золотистый)
- Details: `#8b6f47` (бронзовый)
- Light: `#ffcc66` (теплый желтый)

### Ключевые элементы
- **Шестерни** (большие, средние, малые) - вращающиеся механизмы
- **Маятники** - качающиеся элементы
- **Часовые эмблемы** - пульсирующие источники света
- **Пружины и цепи** - механические детали
- **Пар и искры** - эффекты частиц

### Освещение
- Ambient light: 0.35
- Shadow intensity: 0.65
- Warm tint: 0.4

### Звуки
- Вращение шестерен
- Тиканье часов
- Шипение пара

## Использование в GameMaker Studio 2

### Импорт конфигураций

Каждая пещера имеет JSON-файл конфигурации в:
```
pazzle-game-gamemaker/assets/sprites/cave{N}_{theme}/cave{N}_assets_config.json
```

### Структура конфигурации

Каждый файл содержит:
- `cave_id` - идентификатор пещеры
- `level` - номер уровня
- `theme` - название темы
- `color_palette` - цветовая схема
- `background` - фоновый спрайт
- `furniture` - мебель и крупные объекты
- `decorations` - декоративные элементы
- `effects` - визуальные эффекты и частицы
- `ambient_sounds` - фоновые звуки
- `lighting` - параметры освещения

### Создание спрайтов

Для каждого элемента в конфигурации создайте спрайт в GameMaker Studio 2:

1. **Create > Sprite**
2. Установите размер согласно `size` в конфигурации
3. Для анимированных спрайтов создайте нужное количество фреймов
4. Установите FPS согласно конфигурации
5. Настройте origin (обычно Center для декораций)
6. Для источников света добавьте соответствующую логику в объекты

### Размещение в комнатах

При создании комнат используйте слои согласно конфигурации:
- `Background` - фоновый слой
- `Decorations_Floor` - декорации на полу
- `Decorations_Walls` - декорации на стенах
- `Furniture` - мебель и крупные объекты
- `Effects_Background` - эффекты заднего плана
- `Effects_Foreground` - эффекты переднего плана
- `Effects_Particles` - системы частиц

### Освещение

Для объектов с `light_source: true`:
1. Создайте объект-источник света
2. В событии Draw используйте `draw_set_color()` с цветом из `light_color`
3. Используйте `light_radius` для определения радиуса свечения
4. Примените blend mode для эффектов свечения

## Рекомендации по разработке

1. **Консистентность**: Придерживайтесь цветовой палитры каждой пещеры
2. **Производительность**: Используйте sprite batching для повторяющихся элементов
3. **Анимация**: Синхронизируйте FPS анимаций с общим темпом игры
4. **Звук**: Балансируйте громкость ambient sounds, чтобы не перекрывать музыку
5. **Освещение**: Используйте аддитивный blend mode для эффектов свечения

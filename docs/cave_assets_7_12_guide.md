# Тематические ассеты уровней 7-12

Этот документ описывает тематические ассеты для уровней 7-12 в GameMaker 2 версии игры.

## Обзор

Уровни 7-12 представляют вторую половину текущего 12-пещерного маршрута с уникальными тематиками и повышенной визуальной сложностью.

## Уровень 7: Загадки Сфинкса (Riddle)

**Тема**: `sphinx_wisdom`  
**Головоломка**: Текстовые загадки

### Цветовая палитра
- Primary: `#2a1810` (темно-коричневый)
- Accent: `#d4a574` (песочный)
- Details: `#8b7355` (бежевый)
- Light: `#f4d03f` (золотой)

### Ключевые элементы
- **Статуя сфинкса** - центральный элемент (уникальный)
- **Иероглифы** - светящиеся источники света
- **Древние свитки** - декоративные элементы
- **Золотистая пыль** - эффекты частиц

## Уровень 12: Финальный Подвиг (Epic Finale)

**Тема**: `ultimate_nexus`  
**Головоломка**: Комбинированное испытание

### Цветовая палитра
- Primary: `#0a0a1a` (почти черный)
- Accent: `#ff0066` (ярко-красный)
- Details: `#3a1a2a` (темно-красный)
- Light: `#ff3399` (розово-красный)

### Ключевые элементы
- **Ядро нексуса** - центральный элемент (192x256, уникальный)
- **Энергетические проводники** - передача силы
- **Парящие осколки** - орбитальное движение
- **Энергетический шторм** - интенсивные эффекты
- **Дуги молний** - частые случайные эффекты

### Особенности освещения
- Ambient light: 0.2 (очень темно)
- Shadow intensity: 0.85 (глубокие тени)
- Red tint: 0.6 (сильный красный оттенок)
- Pulsing: true (пульсация 0.6)
- Dynamic intensity: true (динамическая интенсивность)

```
├── cave11_song/cave11_assets_config.json
└── cave12_epic/cave12_assets_config.json
```

# План создания тематических ассетов для ранних пещер 1-2

**Дата создания**: 16.04.2026  
**Статус**: В разработке

## Цель
Создать уникальные визуальные ассеты для ранних пещер (уровни 1-2), чтобы они имели собственную атмосферу и не выглядели однообразно по сравнению с поздними пещерами (3-12).

## Текущее состояние

### Уровень 1: Лабиринт (rm_cave_maze)
- **Текущие ассеты**: Базовый фон (spr_background), стены
- **Тематика**: Исследование, загадочность, эхо
- **Цветовая палитра**: Темно-синие, серые тона
- **Особенности**: Скользящее движение, процедурная генерация

### Уровень 2: Архив/Поиск слов (rm_cave_archive)
- **Текущие ассеты**: Базовый фон (spr_background), сетка букв
- **Тематика**: Древние знания, библиотека, руны
- **Цветовая палитра**: Коричневые, бежевые, золотистые тона
- **Особенности**: Двухфазная структура (поиск слов + память)

## Необходимые ассеты

### Для Уровня 1: Лабиринт

#### Фоновые элементы
1. **spr_cave1_background** - основной фон пещеры
   - Размер: 800x600
   - Стиль: Темная пещера с эхом
   - Элементы: Сталактиты, влажные стены, отражения

2. **spr_cave1_wall** - текстура стен лабиринта
   - Размер: 32x32 (тайл)
   - Варианты: Угловые, прямые, T-образные
   - Детали: Трещины, мох, капли воды

3. **spr_cave1_floor** - текстура пола
   - Размер: 32x32 (тайл)
   - Стиль: Каменный пол с лужами
   - Детали: Неровности, отражения

#### Декоративные элементы
4. **spr_cave1_stalactite** - сталактиты (свисают сверху)
   - Размеры: 16x32, 24x48, 32x64 (разные размеры)
   - Анимация: Капли воды (опционально)

5. **spr_cave1_stalagmite** - сталагмиты (растут снизу)
   - Размеры: 16x24, 24x32, 32x48

6. **spr_cave1_crystal** - светящиеся кристаллы
   - Размер: 16x16, 24x24
   - Анимация: Пульсация света (2-4 фрейма)
   - Цвет: Синий/голубой свет

7. **spr_cave1_water_puddle** - лужи воды
   - Размеры: 32x16, 48x24, 64x32
   - Анимация: Рябь (опционально)

#### Эффекты
8. **spr_cave1_fog** - туман/дымка
   - Размер: 128x128
   - Анимация: Плавное движение (4-6 фреймов)
   - Прозрачность: 30-50%

9. **spr_cave1_echo_effect** - визуальный эффект эха
   - Размер: 64x64
   - Анимация: Расширяющиеся круги (4 фрейма)
   - Использование: При движении игрока

### Для Уровня 2: Архив

#### Фоновые элементы
1. **spr_cave2_background** - фон библиотеки/архива
   - Размер: 800x600
   - Стиль: Древняя библиотека в пещере
   - Элементы: Полки с книгами, свитки, факелы

2. **spr_cave2_bookshelf** - книжные полки
   - Размер: 64x96, 96x128
   - Варианты: Заполненные, частично пустые
   - Детали: Старые книги, свитки, пыль

3. **spr_cave2_floor** - каменный пол с рунами
   - Размер: 32x32 (тайл)
   - Стиль: Гладкий камень с древними символами
   - Детали: Руны, узоры, трещины

#### Декоративные элементы
4. **spr_cave2_torch** - настенный факел
   - Размер: 16x32
   - Анимация: Пламя (4-6 фреймов)
   - Свет: Теплый оранжевый

5. **spr_cave2_scroll** - древние свитки
   - Размеры: 16x24, 24x32
   - Варианты: Свернутые, развернутые
   - Детали: Старая бумага, печати

6. **spr_cave2_rune_stone** - камни с рунами
   - Размер: 32x32, 48x48
   - Варианты: Разные руны
   - Анимация: Слабое свечение (опционально)

7. **spr_cave2_ancient_tablet** - древние таблички
   - Размер: 48x64
   - Стиль: Каменные таблички с текстом
   - Детали: Трещины, выцветший текст

8. **spr_cave2_lectern** - подставка для книг
   - Размер: 32x48
   - Стиль: Деревянная или каменная
   - Детали: Открытая книга сверху

#### Эффекты
9. **spr_cave2_dust_particles** - частицы пыли
   - Размер: 64x64
   - Анимация: Плавающие частицы (6-8 фреймов)
   - Использование: Атмосферный эффект

10. **spr_cave2_glow_rune** - светящиеся руны
    - Размер: 16x16, 24x24
    - Анимация: Пульсация (3-4 фрейма)
    - Цвет: Золотистый/янтарный

11. **spr_cave2_letter_highlight** - подсветка букв в головоломке
    - Размер: 32x32
    - Анимация: Появление/исчезновение (2-3 фрейма)
    - Цвет: Золотистый контур

## Технические требования

### Формат файлов
- **Формат**: PNG с прозрачностью (32-bit)
- **Разрешение**: Кратное 16 пикселям для тайлов
- **Цветовая глубина**: 32-bit RGBA

### Размеры
- **Тайлы**: 16x16, 32x32, 48x48
- **Декор**: Произвольный, но кратный 8 пикселям
- **Фоны**: 800x600 или больше

### Анимация
- **Частота кадров**: 8-12 FPS для плавных анимаций
- **Количество фреймов**: 2-8 в зависимости от эффекта
- **Формат**: Sprite strip (горизонтальная полоса)

### Оптимизация
- **Размер файла**: Максимум 100 КБ на спрайт
- **Компрессия**: PNG оптимизация
- **Атласы**: Группировать похожие спрайты

### Обновление конфигураций комнат

#### rm_cave_maze_config.txt
```json
"layers": [
  {
    "name": "Effects",
    "type": "gmlayer",
    "sprites": ["spr_cave1_fog", "spr_cave1_echo_effect"]
  },
  {
    "name": "Decorations",
    "type": "gmlayer",
    "sprites": ["spr_cave1_stalactite", "spr_cave1_crystal"]
  },
  {
    "name": "Background",
    "imageId": "spr_cave1_background"
  }
]
```

#### rm_cave_archive_config.txt
```json
"layers": [
```

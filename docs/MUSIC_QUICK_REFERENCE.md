## Группы по драматургии

### Ранние (1-3): Исследование
- Ambient, минимализм
- Темп: 70-110 BPM
- Громкость: 0.5-0.65

### Средние (4-6): Нарастание
- Динамика, энергия
- Темп: 95-150 BPM
- Громкость: 0.6-0.8

### Новые (7-9): Вызов
- Разнообразие, сложность
- Темп: 85-135 BPM
- Громкость: 0.6-0.75

### Поздние (10-12): Кульминация
- Интенсивность, эпичность
- Темп: 90-160 BPM
- Громкость: 0.65-0.85

## Интеграция в GameMaker Studio 2

```gml
// В Room Creation Code
switch(room) {
    case rm_cave_maze:
        audio_play_sound(snd_cave_early_1, 10, true);
        audio_sound_gain(snd_cave_early_1, 0.6, 0);
        break;
    case rm_cave_archive:
        audio_play_sound(snd_cave_early_2, 10, true);
        audio_sound_gain(snd_cave_early_2, 0.5, 0);
        break;
    // ... и так далее для всех уровней
}
```

## Контекстные модификаторы

Используйте `scr_audio_manager.gml` для динамической настройки:

```gml
// Во время головоломки
set_music_context("puzzle_active", 0.8);

// Во время диалога
set_music_context("dialogue", 0.4);

// В напряженный момент
set_music_context("intense_moment", 1.0);
```

## Кроссфейд между темами

```gml
// Плавный переход между уровнями
crossfade_music(snd_cave_early_1, snd_cave_early_2, 2.0);
```

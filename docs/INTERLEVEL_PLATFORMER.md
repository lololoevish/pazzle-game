# НОВАЯ МЕХАНИКА: Межуровневый Платформер

## Обзор
Добавлена новая система перемещения между уровнями в виде платформера-хаба, позволяющая игроку перемещаться между различными уровнями игры с помощью физических переходов.

## Основные изменения

### 1. Межуровневый Хаб
- Создана новая комната `rm_interlevel_hub` - большой платформенный мир, соединяющий все 12 уровней
- Игрок может перемещаться между уровнями, прыгая по платформам и используя порталы
- Уровни размещены пространственно, отражаю логику прогрессии игры

### 2. Улучшенные механики перемещения
- Игрок может использовать физическое перемещение между уровнями, а не только меню-навигацию
- Система проверяет разблокировку уровней перед разрешением перехода
- Добавлены визуальные индикаторы доступности уровней

### 3. Расширение до 12 уровней
- Все системы обновлены до поддержки 12 уровней вместо 6
- Сохранения и прогресс учитывают все 12 уровней

### 4. Порталы перехода
- Каждый уровень представлен порталом в межуровневом хабе
- Порталы визуально обозначают тип уровня
- Система проверяет доступность уровня перед переходом

## Технические изменения

### Новые файлы:
- `scr_interlevel_platformer.gml` - основная логика межуровневого перемещения
- `rm_interlevel_hub_config.txt` - конфигурация хаба

### Обновленные файлы:
- `obj_player.gml` - добавлена логика перехода через порталы
- `scr_game_state.gml` - расширение до 12 уровней
- `obj_game_manager.gml` - интеграция с системой межуровневого перемещения

## Игровой процесс
1. Игрок начинает в городе-хабе
2. Может переместиться в межуровневый хаб для выбора уровня
3. В межуровневом хабе расположены 12 порталов, по одному на каждый уровень
4. Игрок может перемещаться между уровнями в зависимости от прогресса
5. После завершения уровня, игрок возвращается в хаб

# Руководство по звуковой драматургии

## Обзор

Звуковая система игры построена на принципах динамической адаптации музыки и звуковых эффектов к игровому контексту. Система поддерживает приоритеты, плавные переходы и контекстные модификаторы громкости.

## Архитектура системы

### Основные компоненты

1. **scr_audio_manager.gml** - Главный менеджер аудио
2. **Система приоритетов** - Управление конфликтами музыкальных тем
3. **Кроссфейд** - Плавные переходы между треками
4. **Контекстные модификаторы** - Динамическая регулировка громкости

### Приоритеты музыки

Система использует 4 уровня приоритета:

- **0 - Room Music**: Музыка комнаты (по умолчанию)
- **1 - Contextual**: Контекстная музыка (puzzle_active, dialogue)
- **2 - Emotional**: Эмоциональные темы (friendly_encounter, mercy_theme)
- **3 - Override**: Принудительная смена (cutscenes, boss fights)

## Использование API

### Базовые функции

#### Воспроизведение музыки
```gml
// Простое воспроизведение
play_music(snd_menu_bg);

// С приоритетом
play_music_with_priority(snd_music_friendly_encounter, 2); // Эмоциональный

// С приоритетом и таймером (автоматически вернется к музыке комнаты)
play_music_with_priority(snd_music_mercy_theme, 2, 30); // 30 секунд

// Принудительная смена (наивысший приоритет)
force_music_change(snd_boss_music, 60); // 60 секунд
```

#### Управление громкостью
```gml
// Установка громкости
set_volume("music", 0.7);
set_volume("sfx", 0.9);
set_volume("master", 1.0);

// Плавное изменение
lerp_volume("music", 0.5, 2.0); // До 0.5 за 2 секунды
```

#### Контекстные модификаторы
```gml
// Применение контекста
set_audio_context("puzzle_active"); // Музыка тише, SFX нормально
set_audio_context("dialogue"); // Музыка очень тихо, SFX тихо
set_audio_context("victory"); // Музыка громко, SFX громко
```

#### Кроссфейд
```gml
// Плавный переход
crossfade_to_music(snd_cave_mid_5, 1.5); // 1.5 секунды

// С callback
crossfade_to_music_advanced(snd_victory_bg, 2.0, function() {
    show_debug_message("Музыка сменилась!");
});
```

### Эмоциональные состояния

```gml
// Установка эмоционального состояния
set_emotional_music_state("friendly_encounter"); // Дружеская встреча
set_emotional_music_state("mercy_theme"); // Тема пощады
set_emotional_music_state("peaceful_resolution"); // Мирное разрешение
set_emotional_music_state("neutral"); // Возврат к музыке комнаты
```

### Система приоритетов

```gml
// Проверка текущего приоритета
var priority = get_current_music_priority();

// Сброс приоритета (возврат к музыке комнаты)
clear_music_priority();

// Обновление системы (вызывать в Step Event)
update_music_priority();
```

## Музыкальные темы по уровням

### Ранние пещеры (1-3)
- **Уровень 1**: `snd_cave_early_1` - Ambient, эхо (громкость 0.6)
- **Уровень 2**: `snd_cave_early_2` - Спокойная, библиотечная (0.5)
- **Уровень 3**: `snd_cave_early_3` - Ритмичная, паттерны (0.65)

### Средние пещеры (4-6)
- **Уровень 4**: `snd_cave_mid_4` - Игривая, зеркальная (0.6)
- **Уровень 5**: `snd_cave_mid_5` - Динамичная, энергичная (0.75)
- **Уровень 6**: `snd_cave_mid_6` - Напряженная, эпичная (0.8)

### Новые пещеры (7-9)
- **Уровень 7**: `snd_cave_new_7` - Древняя, мудрость (0.65)
- **Уровень 8**: `snd_cave_new_8` - Музыкальная, гармоничная (0.7)
- **Уровень 9**: `snd_cave_new_9` - Быстрая, прыжковая (0.75)

### Поздние пещеры (10-12)
- **Уровень 10**: `snd_cave_late_10` - Медитативная, фокус (0.55)
- **Уровень 11**: `snd_cave_late_11` - Хоральная, резонанс (0.7)
- **Уровень 12**: `snd_cave_finale_12` - Героическая, триумф (0.9)

## Контекстные модификаторы

### Neutral (нейтральный)
- Музыка: 100%
- SFX: 100%

### Menu (меню)
- Музыка: 80%
- SFX: 100%

### Town (город)
- Музыка: 70%
- SFX: 90%

### Puzzle Active (активная головоломка)
- Музыка: 50% (не отвлекает)
- SFX: 100%

### Puzzle Complete (завершение головоломки)
- Музыка: 90%
- SFX: 120% (усиление эффекта)

### Dialogue (диалог)
- Музыка: 40% (фокус на тексте)
- SFX: 80%

### Intense Moment (напряженный момент)
- Музыка: 90%
- SFX: 110%

### Victory (победа)
- Музыка: 100%
- SFX: 120%

### Пример 1: Смена комнаты

```gml
if (room_music != undefined) {
    crossfade_to_music(room_music, 1.5);
}
```

### Пример 2: Начало головоломки
```gml

```

### Пример 3: Диалог с NPC

```gml

```

## Best Practices

### 1. Используйте приоритеты правильно
- Музыка комнаты (0) - для обычного геймплея
- Эмоциональные темы (2) - для важных моментов
- Override (3) - только для критических событий

### 2. Всегда используйте кроссфейд
```gml
// Плохо
stop_music();
play_music(new_music);

// Хорошо
crossfade_to_music(new_music, 1.5);
```

### 3. Применяйте контексты
```gml
// При входе в головоломку
set_audio_context("puzzle_active");

// При выходе
set_audio_context("neutral");
```

### 4. Обновляйте систему приоритетов
```gml
// В Step Event главного контроллера или obj_game_manager
update_music_priority();
```

### 5. Используйте таймеры для временных тем
```gml
// Тема проиграет 30 секунд и автоматически вернется
play_music_with_priority(snd_special_theme, 2, 30);
```

## Технические требования

### Форматы файлов
- **Музыка**: OGG Vorbis, 44.1kHz, стерео, качество 6-7
- **SFX**: WAV, 44.1kHz, моно/стерео, 16-bit

### Размеры
- Музыкальная тема: 1-2 МБ (2-3 минуты loop)
- Звуковой эффект: 10-50 КБ

### Производительность
- Максимум 2 музыкальных трека одновременно (кроссфейд)
- До 8 одновременных SFX
- Предзагрузка музыки для текущего и следующего уровня

## Troubleshooting

### Музыка не меняется
- Проверьте приоритет текущей музыки: `get_current_music_priority()`
- Убедитесь, что новая музыка имеет достаточный приоритет
- Используйте `force_music_change()` для принудительной смены

### Громкость не применяется
- Проверьте контекстные модификаторы: `audio_config.current_context`
- Убедитесь, что вызываете `set_audio_context()` после смены контекста

### Кроссфейд не работает
- Убедитесь, что оба звука существуют
- Проверьте, что звуки загружены в память
- Используйте `is_audio_available()` для проверки доступности системы

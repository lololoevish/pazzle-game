# Конфигурация музыкальных тем для всех уровней

## Структура музыкальной драматургии

### Ранние пещеры (1-3): Исследование и загадочность

#### Уровень 1: Лабиринт со скольжением
- **Файл**: `snd_cave_early_1`
- **Настроение**: Ambient, эхо, минимализм
- **Темп**: 80-90 BPM
- **Инструменты**: Синтезаторные пады, реверберация, редкие перкуссии
- **Описание**: Создает ощущение пустоты и неизвестности. Звуки эха подчеркивают механику скольжения.
- **Громкость**: 0.6 (тихая, атмосферная)

#### Уровень 2: Поиск слов
- **Файл**: `snd_cave_early_2`
- **Настроение**: Спокойная, размышляющая, библиотечная
- **Темп**: 70-80 BPM
- **Инструменты**: Акустическая гитара, колокольчики, тихие струнные
- **Описание**: Мягкая мелодия для концентрации. Ассоциируется с древними знаниями.
- **Громкость**: 0.5 (очень тихая для концентрации)

#### Уровень 3: Паттерн-память
- **Файл**: `snd_cave_early_3`
- **Настроение**: Ритмичная, паттерны, механическая
- **Темп**: 100-110 BPM
- **Инструменты**: Синтезированные ритмы, повторяющиеся мотивы
- **Описание**: Ритмичная структура помогает запоминать паттерны. Часовые мотивы.
- **Громкость**: 0.65

---

### Средние пещеры (4-6): Нарастание напряжения

#### Уровень 4: Поиск пар карточек
- **Файл**: `snd_cave_mid_4`
- **Настроение**: Игривая, концентрация, зеркальная
- **Темп**: 95-105 BPM
- **Инструменты**: Ксилофон, маримба, легкие перкуссии
- **Описание**: Легкая, но требующая внимания. Зеркальные мотивы в мелодии.
- **Громкость**: 0.6

#### Уровень 5: Платформер со сбором кристаллов
- **Файл**: `snd_cave_mid_5`
- **Настроение**: Динамичная, энергичная, кристаллическая
- **Темп**: 120-130 BPM
- **Инструменты**: Электронные биты, синтезаторы, звон кристаллов
- **Описание**: Быстрая и энергичная для поддержки платформенного геймплея.
- **Громкость**: 0.75

#### Уровень 6: Финальное испытание на время
- **Файл**: `snd_cave_mid_6`
- **Настроение**: Напряженная, эпичная, кульминационная
- **Темп**: 140-150 BPM
- **Инструменты**: Оркестровые ударные, струнные, медные духовые
- **Описание**: Нарастающее напряжение. Ощущение финальной битвы первой части.
- **Громкость**: 0.8

---

### Новые пещеры (7-9): Мистика и испытания

#### Уровень 7: Загадки Сфинкса
- **Файл**: `snd_cave_new_7`
- **Настроение**: Древняя, мудрость, египетская
- **Темп**: 75-85 BPM
- **Инструменты**: Этнические инструменты, флейты, ситар
- **Описание**: Восточные мотивы. Ощущение древней мудрости и загадок.
- **Громкость**: 0.65

#### Уровень 8: Звуковые Ловушки
- **Файл**: `snd_cave_new_8`
- **Настроение**: Музыкальная, гармоничная, резонансная
- **Темп**: 90-100 BPM
- **Инструменты**: Арфа, колокола, гармонические обертоны
- **Описание**: Чистые тона и гармонии. Музыка как часть головоломки.
- **Громкость**: 0.7

#### Уровень 9: Прыгающий Путь
- **Файл**: `snd_cave_new_9`
- **Настроение**: Быстрая, прыжковая, ритмичная
- **Темп**: 130-140 BPM
- **Инструменты**: Синкопированные ритмы, электронные басы
- **Описание**: Акцент на ритме прыжков. Синкопы подчеркивают тайминг.
- **Громкость**: 0.75

---

### Поздние пещеры (10-12): Кульминация

#### Уровень 10: Запоминалка (продвинутая память)
- **Файл**: `snd_cave_late_10`
- **Настроение**: Медитативная, фокус, глубокая
- **Темп**: 60-70 BPM
- **Инструменты**: Тибетские поющие чаши, дроны, минимальная перкуссия
- **Описание**: Глубокая концентрация. Медитативное состояние для сложной памяти.
- **Громкость**: 0.55

#### Уровень 11: Песнь Пещер
- **Файл**: `snd_cave_late_11`
- **Настроение**: Хоральная, резонанс, пещерная акустика
- **Темп**: 80-90 BPM
- **Инструменты**: Хор, реверберация, органные басы
- **Описание**: Голоса эхом отражаются от стен. Ощущение живой пещеры.
- **Громкость**: 0.7

#### Уровень 12: Финальный Подвиг
- **Файл**: `snd_cave_finale_12`
- **Настроение**: Героическая, триумф, эпическая
- **Темп**: 150-160 BPM
- **Инструменты**: Полный оркестр, хор, мощные ударные
- **Описание**: Кульминация всего путешествия. Героический финал.
- **Громкость**: 0.9

## Технические параметры

### Формат файлов
- **Музыка**: OGG Vorbis, 44.1kHz, стерео, качество 6-7
- **SFX**: WAV, 44.1kHz, моно/стерео, 16-bit

### Размеры
- Музыкальная тема: 1-2 МБ (2-3 минуты loop)
- Звуковой эффект: 10-50 КБ

### Loop points
- Все музыкальные темы должны иметь бесшовный loop
- Точки loop устанавливаются в метаданных OGG

## Переходы между темами

### Правила кроссфейда
1. **Смена комнаты**: 1.5 секунды
2. **Смена контекста**: 0.5 секунды
3. **Эмоциональное событие**: 2.0 секунды
4. **Пауза/возобновление**: 0.3 секунды

### Приоритеты
1. Эмоциональные темы (высший)
2. Музыка комнаты
3. Контекстные модификаторы
4. Базовая громкость

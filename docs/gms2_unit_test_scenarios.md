# Документ сценариев модульного тестирования для GMS2 версии

## Обзор

Этот документ содержит конкретные сценарии для модульного тестирования компонентов GameMaker Studio 2 версии "Adventure Puzzle Game". Каждый сценарий представляет собой независимый модульный тест, который проверяет конкретную функцию или метод системы.

## Структура тестов

Каждый сценарий тестирования состоит из:
- Цель теста
- Предусловия
- Шаги выполнения
- Ожидаемый результат
- Фактический результат
- Статус (пройден/провален)

## Расписание тестирования

### Еженедельный цикл:
- **Понедельник:** Модульное тестирование новых функций
- **Среда:** Интеграционное тестирование
- **Пятница:** Функциональное тестирование и регрессия

### При каждом билде:
- Запуск всех модульных тестов
- Проверка основного игрового цикла
- Тестирование сохранений

## Ответственные

- **Разработчик:** Написание и поддержка unit-тестов
- **QA инженер:** Проведение интеграционного и функционального тестирования
- **Тестировщик:** Ручное тестирование и проверка юзабилити
